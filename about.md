## 目的

*Markdown 101* 是为新手和专家设计的综合 Markdown 参考。它源于对现有的 Markdown 引用不完整、不充分或两者兼而有之的挫败感。

## 贡献

这是一个开源项目，欢迎您的贡献。该存储库托管在 [GitHub](https://github.com/mattcone/markdown-guide) 上。有关说明和指南，请参阅 [README](https://github.com/mattcone/markdown-guide/blob/master/README.md) 。

## 报告问题

发现错字或不准确的陈述？请在 GitHub 项目中创建一个 [issue](https://github.com/mattcone/markdown-guide/issues) 。
